# Calculator Project

## Description

This is a simple calculator built using HTML, CSS (Tailwind), and JavaScript.

## Features

* Basic operations (Add, Subtract, Multiply, Divide)
* Responsive design
* Clean UI

## Live Demo

# Technologies Used

* HTML
* Tailwind CSS
* JavaScript

##  How to Use

1. Open the website
2. Click buttons to perform calculations
